**Usage guidelines:**

- Pre-processing: normalization + SentencePiece (spm32k,spm32k)
- A sentence-initial target-language token is required, in the form `>>id<<` (id = valid target language ID, e.g. `>>cmn_Hans<<`)
- Load via Transformers: `pipeline("translation", model="Helsinki-NLP/opus-mt-en-zh")`, or directly with `AutoTokenizer` + `AutoModelForSeq2SeqLM`. (Note: the "translation" pipeline type requires `transformers<5.0.0`; otherwise load the model directly.)
