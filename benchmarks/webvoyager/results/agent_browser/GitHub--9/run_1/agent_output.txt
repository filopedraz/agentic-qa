- **Purpose:** A command-line tool that takes a PDF and deliberately degrades it to look like a physical scan of a printout — applying skew, grayscale, warm paper tone, scanner grain, defocus, edge shadow, and JPEG compression artifacts. Each page is rasterized, run through the effect pipeline, and reassembled into a new image-only PDF (selectable text is removed, faithful to a real scanner). Output is deterministic by default. It also runs entirely client-side in the browser via WebAssembly (using PDF.js for rasterization).
