Main features: support for multiple instruments including 5-fret guitar, drums (plastic controllers or electronic kits), vocals (microphone), pro-guitar and pro-keys; cross-platform play; custom song/chart support via the YARG Setlist and chart formats (GuitarGame_ChartFormats); MIDI input handling (DryWetMIDI, MIDI Input for Unity); and installation/management through the YARC Launcher.
